固定修改紀錄檔
之後此專案的所有修改都記錄在這裡。

2026-06-16

1. 建立 Godot 4 俯視角生存遊戲原型
- 建立 project.godot、主場景、玩家、敵人、掉落物場景與腳本。
- 玩家可 WASD 移動。
- 玩家會自動攻擊最近敵人。
- 敵人會從畫面外生成並追蹤玩家。
- 敵人碰到玩家會造成傷害。
- 敵人死亡會掉落經驗值與金錢。
- 玩家靠近掉落物可自動撿取。
- UI 顯示生命、等級、經驗、金錢。
- 美術先使用簡單圓形、色塊與線條。

2. 安裝 Godot 4
- 下載並解壓 Godot 4.6.3 stable official 到 tools/Godot_4.6.3。
- 使用 Godot 命令列版本驗證專案可載入。

3. 加入升級系統
- 玩家取得經驗後可升級。
- 升級時暫停遊戲。
- 顯示三個隨機升級選項。
- 選擇後套用能力並恢復遊戲。
- 升級項目包含傷害、攻速、移速、攻擊範圍、最大生命、金錢掉落率。

4. 加入 Slot Game 系統
- 地圖每 90 秒生成 Slot 機。
- 玩家靠近 Slot 機會自動開啟 Slot UI。
- Slot UI 開啟時暫停遊戲。
- 保留三種機台：50 元、150 元、400 元。
- 保留原本機率、無空獎、金錢不足不可抽、幸運值保底規則。
- 顯示目前金錢、幸運值、三個機台按鈕、三個轉輪圖示、獎勵名稱與效果。

5. 繁中化與戰鬥顯示
- 遊戲內文字改為繁中顯示。
- 攻擊怪物時，目標頭上顯示傷害數字。
- Slot 機改成靠近自動開啟，不需要按 E。
- Slot UI 關閉後要離開範圍再靠近才會再次開啟。
- 升級系統數值改為：傷害 +15%、攻速 +15%、移動速度 +8%、攻擊範圍 +10%、最大生命 +20、金錢掉落率 +10%。

6. Slot 獎池更新
- 安慰獎改為小補血 50 生命、小經驗 50 經驗。
- 普通獎改為傷害 +30%、攻速 +30%、移動速度 +15%、撿取範圍 +30%。
- 稀有獎改為落雷、刺環、彈射、多重。
- 大獎改為骰子分裂、777爆炸、金錢護盾、招財貓。
- 落雷每 2 秒攻擊可視範圍最多 2 隻怪物，各造成 100 傷害。
- 刺環跟隨玩家，範圍內怪物每秒受到 30 傷害。
- 彈射可累加，每次增加 1 次額外彈射。
- 多重可累加，每次增加 2 個額外投射物。
- 骰子分裂可累加，命中後分裂成 2 顆小骰子，小骰子造成 50% 傷害。
- 777爆炸改為每第 3 次攻擊觸發，以命中敵人為中心造成範圍爆炸。
- 金錢護盾改為每 60 秒金錢能量攻擊可視範圍所有怪物，不消耗金錢。
- 招財貓改為實體寵物，會跟隨玩家、優先撿錢、並用玩家目前傷害 50% 攻擊敵人；重複抽到會增加一隻。

7. 選單暫停、勝利條件與節奏調整
- 任何升級選單或 Slot UI 開啟時，都使用全域暫停，讓怪物、掉落物、寵物、計時器與地圖物件停止動作。
- 怪物強化速度放慢為原本 2 倍間隔，敵人能力倍率由每 100 秒增加改為每 200 秒增加。
- Slot 機生成間隔由每 90 秒改為每 60 秒。
- 遊戲時間設定為 10 分鐘，撐過 10 分鐘即過關。
- 玩家死亡或過關後會停止遊戲，顯示結果文字，按 Space 可重新開始。
- 角色每次升級時會恢復最大生命 20%。
- UI 新增遊戲時間顯示。

8. 廢棄晶片、存檔欄位與結算機制
- 新增 money、chip_pickups、total_chips 的資源規則。
- money 維持單局用途，可用於 Slot，死亡結束時歸零。
- chip_pickups 為本局撿到的廢棄晶片數。
- total_chips 為永久廢棄晶片，會寫入 user://save_slots.json 保存。
- 新增晶片掉落物，玩家靠近會自動吸取，撿取後 chip_pickups 增加，並顯示晶片飄字與簡易音效。
- 普通怪 5% 機率掉 1 個晶片。
- 菁英怪必掉 3 到 5 個晶片。
- 小 BOSS 必掉 10 個晶片。
- 關卡 BOSS 必掉 30 個晶片。
- 新增菁英怪、小 BOSS、關卡 BOSS 的簡易生成與類型標記。
- UI 顯示金錢、本局晶片、永久晶片。
- 死亡或過關結算時顯示生存時間、擊殺數、BOSS 傷害、本局獲得晶片，並將本局晶片加入永久晶片後存檔。
- 新增主選單：開新遊戲、讀取存檔、設定、離開遊戲。
- 新增 5 個存檔欄位，顯示帳號等級、遊玩小時、最後遊玩時間、永久晶片。
- 開新遊戲選到已有資料的欄位時，會出現覆蓋確認。
- 新增設定頁：音量、全螢幕、按鍵說明。
- 修正暫停根因：主場景維持 UI 可操作，但玩家、敵人、掉落物、寵物與生成 Timer 皆明確設為可暫停，確保升級選單、Slot UI、死亡結算、設定、主選單開啟時，遊戲物件與遊戲時間都停止。

- 晶片掉落改為只有菁英怪與 BOSS 會掉落：菁英怪 50% 掉 1 顆，BOSS 每 60 秒生成且必掉 5 顆。
- 掉落物吸向玩家速度提升 1.5 倍。

- 保留死亡與勝利結算、存檔晶片累積，以及遊戲時間 10 分鐘勝利條件。

16. 遊戲經濟系統重構：Slot 代幣版
- 資源重新定位為三種：Money 金幣用於本局基礎能力升級；Slot Token 用於轉 Slot；Chip 廢棄晶片保留作為局外永久研究資源。
- Slot 不再消耗 Money，改為每次 Spin 固定消耗 1 Slot Token；Token 不足時無法轉動，Auto Spin 也只會在有 Token 時運作。
- 新增 Slot Token 掉落：普通怪 6% 掉 1 枚、菁英怪 40% 掉 1 枚、Boss 固定掉 3 枚。
- 原三選一「消耗率」改為「代幣掉落」，每級 +5% Slot Token 掉落率，避免 1 Token 制下技能失效。
- 新增右側常駐金幣升級面板，可在戰鬥中直接花金幣升級，不暫停遊戲。
